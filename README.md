🛡️ Sentinel Omega v0.1.1
Sentinel Omega es una herramienta táctica de seguridad diseñada para entornos educativos. Permite a los usuarios reportar incidentes en tiempo real mientras ofrece una interfaz administrativa para la gestión de crisis y generación de actas legales en PDF.

✨ Características Principales:
Análisis de Riesgo: Motor lingüístico que detecta palabras clave para clasificar emergencias.

Código Rojo: Bloqueo total del sistema ante amenazas letales (Tiroteo/Incendio).

Gestión Documental: Generación automática de actas en PDF y reportes en Excel.

Privacidad: Sistema de purga automática de datos tras 7 días.
